# Triage Labels

Canonical triage roles for this repository:

- `triage:needs-info`: Missing reproduction details, logs, or requirements.
- `triage:confirmed`: Reproduced or validated and ready for planning.
- `triage:planned`: Accepted and sliced into actionable scope.
- `triage:in-progress`: Actively being implemented.
- `triage:blocked`: Cannot proceed due to an external dependency.
- `triage:review`: Implementation complete, awaiting review or validation.
- `triage:done`: Merged, released, or otherwise completed.

If labels do not exist in the repository yet, create them before using triage workflows.
